# ZZZ-Scanner

- 一个驱动盘扫描导出工具
- 相对点位置+计算实现多分辨率适配
- PInvoke 实现游戏窗口操作
- OnnxRuntime+PP-OCRv5 进行文字识别
- 字典+编辑距离算法实现结果纠错

## 功能

- [x] 屏幕适配
- [x] 翻页遍历
- [x] OCR 识别
- [x] 结果纠错

## 使用

1. 下载并解压
2. 打开游戏，设置简体中文+细体+16:9 的分辨率（如 1920x1080，1280x720 等）
3. 打开背包
4. 运行软件 ZZZ-Scanner.exe，会请求管理员权限，用于界面操作
5. 结果会保存到 json 中

- 扫描中按 ESC 退出背包会停止扫描
- 粗体可能影响 OCR 识别准确率
- 其他屏幕比例（如 16:10）可以先全屏游戏记录下点位，然后修改 Resource/config.json 中的配置
