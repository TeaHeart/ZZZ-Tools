# ZZZ-Scanner

> 一个驱动盘扫描导出工具
> 相对点位置+计算实现多分辨率适配
> PInvoke实现游戏窗口操作
> OnnxRuntime+PP-OCRv5进行文字识别
> 字典+编辑距离算法实现结果纠错

## 功能

- [x] 屏幕适配
- [x] 翻页遍历
- [x] OCR识别
- [x] 结果纠错

## 使用

1. 下载并解压
2. 打开游戏，设置简体中文+细体+16:9的分辨率（如1920x1080，1280x720等）
3. 打开背包
4. 运行软件ZZZ-Scanner.exe，会请求管理员权限，用于界面操作
5. 结果会保存到*.json中

- 扫描中按ESC退出背包会停止扫描
- 粗体可能影响OCR识别准确率
- 其他屏幕比例（如16:10）可以先全屏游戏记录下点位，然后修改Resource/config.json中的配置
